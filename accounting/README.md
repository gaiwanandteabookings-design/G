# Реестр невосполнимых затрат — GAIWAN LLC

Учёт невосполнимых (стартовых) затрат на открытие бизнеса: первичные документы,
чеки и номера заказов. Доходы и прибыль не ведутся — только вложения.
Все суммы в **долларах ($)**. По состоянию на 26 июня 2026.

## Файлы

| Файл | Что это |

## Важно про суммы (проверено)

Все 51 позиция сверены с исходными документами. Итог считается автоматически:

- Сумма всех первичных позиций (50 строк): **$60 487.37**
- Аренда за весь срок (54 месяца), имеются копии банковских чеков: **$54 000.00**
- **ИТОГО невосполнимых затрат: $114 487.37**

> Ранее в черновиках стояли итоги **$110 523** и **$62 466** — оба не сходились с
> суммой строк (ошибки ручного подсчёта). В этом файле итог считается формулой,
> поэтому всегда верный.

## Примечания

- Мелкие чеки Home Depot и часть мелких расходов сгруппированы.
- Полные сканы всех документов хранятся у владельца бизнеса.
- Номера заказов/чеков перенесены со скриншотов — при необходимости сверь их
  с оригиналами (на мелком экране возможны опечатки в длинных номерах).
