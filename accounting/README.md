# Бухгалтерский учёт при открытии бизнеса

Базовый комплект учётных документов (регистров) для запуска бизнеса.
Все суммы — в **долларах США ($)**.

> Все таблицы лежат в формате **CSV** — открываются в Excel, Google Sheets,
> Numbers. В каждой таблице первая строка — заголовки, далее **примеры строк**
> (`Sample row`). Заменяй примеры своими данными.

## Что входит в комплект

| Файл | Что это | Когда заполнять |
|------|---------|-----------------|
| `01_startup_investments.csv` | **Реестр первоначальных вложений** — всё, что вложено до и при открытии (оборудование, мебель, ремонт, регистрация, реклама, закупка товара) | На этапе запуска |
| `02_fixed_assets.csv` | **Реестр основных средств** — дорогое имущество, которое служит долго (техника, мебель) | По мере покупки |
| `03_income_expenses.csv` | **Книга учёта доходов и расходов** — основной операционный учёт после запуска | Ежедневно/еженедельно |
| `04_primary_documents.csv` | **Журнал первичных документов** — чеки, накладные, акты как основания для записей | При получении каждого документа |
| `05_cash_flow.csv` | **Движение денежных средств** — приход/расход денег и остаток | По каждой операции |

## Ключевые понятия

- **Первоначальные (стартовые) вложения** — всё, что потрачено на запуск бизнеса.
- **CAPEX (капитальные затраты)** — дорогие вещи, служащие годами (техника, мебель).
- **Inventory (товарные запасы)** — то, что продаётся или расходуется (чай, посуда на продажу).
- **Sunk cost (невозвратные / невосполнимые затраты)** — то, что обратно не вернуть
  (ремонт, регистрация, реклама на старте).
- **Первичный документ** — бумага-основание (чек, накладная, акт), подтверждающая операцию.

## Как считать итоги

CSV не хранит формулы. Открой файл в Excel / Google Sheets и добавь строку итогов:

- Итог по столбцу `Amount (USD)` — общая сумма: `=SUM(G2:G100)`
- Итоги по категориям — сводная таблица (Pivot Table) по столбцу `Category`.

## Связь между таблицами

Столбец `Document` / `Related to` связывает записи с первичными документами
(`04_primary_documents.csv`). Каждая запись о вложении или расходе должна
ссылаться на документ-основание — это и есть «первичный учёт».
